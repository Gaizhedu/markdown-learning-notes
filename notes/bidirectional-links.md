# 双向链接
在Markdown中，可以创建双向链接来链接两个文件

例如：[[index.md]]

通过这样，你可以在一个文件里面快速连接其他的文件

## 双向链接的格式
要想实现双向链接的效果非常简单，只需要用两个方括号将文件包起来即可：`[[文件名称]]`

实例如下：
```
这个双向链接连接到了导航页：[[index.md]]
```
输出效果如下：

这个双向链接连接到了导航页：[[index.md]]
